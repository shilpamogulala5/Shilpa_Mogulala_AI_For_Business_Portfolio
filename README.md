## Portfolio Reflection

Throughout this course, I gained a stronger understanding of how artificial intelligence can be used to solve practical business problems. Before taking this course, I understood AI mainly as a technology that could automate tasks or make predictions. Through the projects, I learned that successful AI work involves much more than simply running a model. It requires defining the problem clearly, preparing reliable data, choosing an appropriate method, evaluating the results, and explaining what the results mean in a business context.

The course introduced me to several major areas of machine learning, including classification, support vector machines, decision trees, dimensionality reduction, artificial neural networks, and natural language processing. Working with these topics helped me see how different algorithms can be useful for different purposes. For example, classification models can help organizations sort records into meaningful categories, decision trees can support understandable decisions, neural networks can identify complex patterns, and NLP can help analyze information contained in written text.

I developed technical skills in Python-based machine learning workflows, data cleaning, visualization, model training, model comparison, and performance evaluation. I also became more comfortable working through model errors, adjusting parameters, reviewing results, and improving an approach when the original outcome was not strong enough. One of the most valuable lessons from this course was learning that a high accuracy score alone does not guarantee that a model is useful. A model also needs to be reliable, understandable, fair, and connected to a real business need.

These projects helped prepare me for workplace situations where data-driven tools may be used to analyze customer behavior, predict future outcomes, automate repetitive work, identify trends, or summarize large amounts of text information. I now feel more confident using machine learning tools as part of a structured problem-solving process. This portfolio reflects both my technical growth and my ability to think about how AI can create value for organizations.
